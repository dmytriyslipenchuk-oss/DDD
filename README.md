# Генератор сертифікатів

Змінюється:
- дата під номером
- статична частина номера
- початковий номер
- кількість цифр
- хвіст номера

Для Vercel:
- Framework Preset: Vite
- Build Command: vite build
- Output Directory: dist


Оновлено: шаблон сертифіката замінено на ГРАНТ у вбудованому base64 script.js.
